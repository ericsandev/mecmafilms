<!-- GETTING STARTED -->
## Getting Started

### Setup .env example

Create .env file in your root project folder.

```
NEXT_PUBLIC_MAIN_URL=http://localhost:3000/
```

### Installation

1. Install NPM packages
```sh
npm install or yarn install
```
2. Run project
```sh
npm run dev or yarn dev
```
